A Modern Search Interface for Oracle EBS

The EBS Component Search Utility (ECSU) provides a fast, secure, and intuitive web interface for developers and administrators to search and view component details across multiple Oracle E-Business Suite instances.

Get Started on GitHub

Key Features

Built for efficiency and security.

Centralized Search

Query multiple Oracle EBS instances from a single, unified search interface.

Secure Authentication

Leverages Keycloak for robust, role-based access control (RBAC).

Dynamic Suggestions

Intelligent, real-time search suggestions help you find components faster.

Role-Based Views

A tailored user experience for Developers and Administrators.

Admin Management

Admins can easily manage user access and configure instance connections.

Modern Tech Stack

Built with a powerful Spring Boot backend and a responsive React frontend.
